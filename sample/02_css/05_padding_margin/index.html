padding と margin

padding と margin は CSS で余白をつくるために使います。似ているけど違う、違うけれど似ているこの2つの違いをしっかりマスターしましょう。

指定なし

なにも指定していません。この場合 margin も padding も 0 になります。

padding を 50px とる

padding を 50px 指定しています。この場合 div の **内側** に50pxの余白とられます。

margin を 50px とる

margin を 50px 指定しています。この場合 div の **外側** に50pxの余白とられます。

margin と padding を 50px とる

margin を 50px 指定しています。この場合 div の **内側と外側** に50pxの余白とられます。
